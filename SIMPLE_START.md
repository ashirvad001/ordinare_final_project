# Ordinare - Simple Start Guide

## Quick Start (1 Step)

**Double-click:** `START_APP.bat`

Or run:
```bash
python app.py
```

Then open: http://127.0.0.1:5000

## Features

✅ **Simple Signup** - Email, username, password
✅ **Secure Login** - Password hashing
✅ **Attendance Tracking** - Mark daily attendance
✅ **Study Timer** - Pomodoro timer with analytics
✅ **Bunk Calculator** - Calculate safe bunks
✅ **Grade Predictor** - AI-powered grade prediction
✅ **Study Optimizer** - ML-based study recommendations
✅ **Timetable** - Weekly schedule management
✅ **Dark Mode** - System-adaptive theme

## Login Methods

1. **Email + Password** - Simple and secure

## No Configuration Needed

- ❌ No email OTP setup required
- ❌ No Google OAuth setup required
- ❌ No environment variables needed
- ✅ Just run and use!
